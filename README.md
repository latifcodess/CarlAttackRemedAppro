## Installation

### 1. Cloner le projet

```bash
git clone https://github.com/latifcodess/CarlAttackRemedAppro.git
cd CarlAttackRemedAppro/CarlAttack
```

### 2. Installer les dépendances (si nécessaire)

```bash
dotnet workload install monogame
```

### 3. Build le projet

```bash
dotnet build
```

### 4. Lancer le projet

```bash
dotnet run
```
